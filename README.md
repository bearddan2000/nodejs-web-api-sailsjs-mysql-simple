# nodejs-web-api-sailsjs-mysql-simple

## Description
A sailsjs api that queries mysql server.

## Tech stack
- sailsjs
- mysql

## Docker stack
- node:latest
- mariadb

## To run
`sudo ./install.sh -u`
curl -i localhost/dogs

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
